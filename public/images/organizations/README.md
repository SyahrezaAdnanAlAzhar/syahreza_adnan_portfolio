# Organization Images Directory

Place organization logos and related images here.

## File Naming Convention

- Use lowercase with hyphens
- Format: `organization-name-logo.png`
- Example: `genbi-telkom-logo.png`, `ai-lab-logo.png`

## Recommended Image Specifications

- **Format**: PNG (preferred for logos with transparency) or JPG
- **Dimensions**: 400x400px minimum
- **File Size**: < 500KB per image
- **Aspect Ratio**: Square (1:1) or standard logo ratio
