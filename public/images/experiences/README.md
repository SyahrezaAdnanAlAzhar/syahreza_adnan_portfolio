# Experience Images Directory

Place company logos and related images for work experiences here.

## File Naming Convention

- Use lowercase with hyphens
- Format: `company-name-logo.png` or `company-name-logo.jpg`
- Example: `ina-digital-logo.png`, `fpt-software-logo.png`

## Recommended Image Specifications

- **Format**: PNG (preferred for logos with transparency) or JPG
- **Dimensions**: 400x400px minimum
- **File Size**: < 500KB per image
- **Aspect Ratio**: Square (1:1) or standard logo ratio
