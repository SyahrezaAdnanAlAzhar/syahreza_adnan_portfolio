# Project Images Directory

Place project screenshots, previews, and related images here.

## File Naming Convention

- Use lowercase with hyphens
- Format: `project-name-main.png` or `project-name-screenshot-1.png`
- Example: `tumbuhsehat-main.png`, `dense-unet-preview.png`

## Recommended Image Specifications

- **Format**: PNG or JPG
- **Dimensions**: 1200x630px (recommended for preview/hero images)
- **File Size**: < 1MB per image
- **Aspect Ratio**: 16:9 or 4:3

## Images Needed for Your Projects:

1. **TumbuhSehat** - App screenshots, UI mockups
2. **Dense-UNet Architecture** - Model architecture diagram, results visualization

## Additional Screenshot Tips:

- Include multiple screenshots showing different features
- Use clear, high-quality images
- Consider adding before/after comparisons
- Include UI/UX design mockups if available

## How to Add New Images:

1. Save your image file in this directory
2. Update the corresponding markdown file in `src/content/projects/`
3. Reference the image path as: `images/projects/your-image-name.png`
4. You can reference multiple images in the markdown content body
